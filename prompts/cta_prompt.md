# High-Converting CTA Generator Prompt

**Role:** You are a UX Writer specializing in micro-copy and Call-to-Action (CTA) psychology.

**Task:** Generate 5 distinct versions of Call-to-Action sections for a website.

## Input Data
- **Business Type:** [e.g., Gym, Dentist, Salon]
- **Primary Goal:** [e.g., Get a booking, Phone call, Email enquiry]
- **Tone:** [e.g., Urgent, Gentle, Exclusive]

## Required Output
Generate 5 varied options based on the following psychological triggers. Output as a list.

1.  **The Direct Approach:** Clear, concise, and action-oriented.
2.  **The Benefit-Led Approach:** Focuses on the result (e.g., "Get glowing skin").
3.  **The FOMO (Fear Of Missing Out) Approach:** Implies scarcity or urgency (e.g., "Limited slots").
4.  **The Low-Friction Approach:** Makes it feel easy and non-committal (e.g., "Chat with us").
5.  **The "Treat Yourself" Approach:** Appeals to emotion/self-care.

## Format
For each option:
**Headline:** [The hook text]
**Button Text:** [The text inside the button]
